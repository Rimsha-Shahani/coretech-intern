# Task 4: Supervised Learning – Regression

## Project Overview
This project builds a regression model to predict project budgets using a client dataset. The workflow includes data loading, preprocessing, feature selection, model training, prediction, evaluation, and comparison of regression models.

## Objectives
- Load and preprocess the dataset
- Select input features and target variable
- Split the data into 80% training and 20% testing sets
- Train a Linear Regression model
- Make budget predictions
- Evaluate the model using MAE, MSE, RMSE, and R² Score
- Train and compare a Ridge Regression model
- Visualize Actual vs Predicted values

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Files Included
- `Task4_Regression_Model.ipynb` – Jupyter notebook containing the complete implementation
- `cleaned_dataset.csv` – Dataset used for training and testing
- `README.md` – Project documentation

## Workflow
1. Import required libraries
2. Load the dataset
3. Explore and preprocess data
4. Select features and target variable
5. Split data into training and testing sets (80/20)
6. Train Linear Regression model
7. Make predictions
8. Evaluate model performance using:
   - Mean Absolute Error (MAE)
   - Mean Squared Error (MSE)
   - Root Mean Squared Error (RMSE)
   - R² Score
9. Train Ridge Regression model
10. Compare both models
11. Plot Actual vs Predicted values

## Results
The regression models successfully predicted project budgets based on the selected features. Performance was evaluated using MAE, MSE, RMSE, and R² Score, and Ridge Regression was compared with Linear Regression to analyze model effectiveness.
